Neptune CRM

This project is a full-stack CRM I designed and built for a recruitment startup using React, SCSS, Node.js/Express, and PostgreSQL. It brings together a polished frontend and a robust backend to streamline lead management.

I implemented role-based access controls, real-time reminders, contact management, and activity tracking. To secure and sync data instantly, I integrated Supabase for authentication and live updates, using JWTs for session safety.

I also led the CI/CD deployment on Render for both frontend and backend—automating builds, ensuring high availability, and architecting the system for scalability, security, and peak performance.
